Multi User Test Runner

Multi User Test Runner is a custom JUnit (4.12+) test runner for testing authorization with multiple different users. Library requires only minimal configuration to be able run tests with many different resource creator and user combinations.

Usage

  1. Create a configured abstract class by extending {@link fi.vincit.multiusertest.test.AbstractUserRoleIT} class and implement methods. This will be your base class for your test class.
  2. Configure runner and default expected annotation with {@link fi.vincit.multiusertest.annotation.MultiUserTestConfig}
  3. Create a test class which is extended from your configured class.
  4. Add {@link fi.vincit.multiusertest.annotation.TestUsers} annotation for your test class
  5. Write test methods
  6. Add e.g. authentication().expect(toFail(ifAnyOf("user:user"))); before method to test to mark which roles/users are expected to fail
